Rural Healthcare Clinic Management System

This project is a PHP + MySQL web application. The interactive dashboard and patient management screens require a server environment and database, so they cannot run directly from GitHub Pages.

Use this repository with Docker locally or deploy it to a PHP-compatible host such as Render, Railway, or Fly.io to access the full application.

View source code Read setup instructions

Local run: docker compose up --build -d

Then open http://127.0.0.1:8080/.
